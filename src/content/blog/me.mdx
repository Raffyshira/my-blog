<h1 class="w-72">Mari kita selami lebih dalam tentang diriku, oke?</h1>

Perjalanan ini dimulai ketika saya lulus dari sekolah menengah atas. Saya belajar di MAN 3 Banyuwangi selama 3 tahun. Saat sedang scroll Instagram Reels di halaman Explore, saya menemukan sebuah video yang menarik minat saya terhadap pengembangan web. Rasa penasaran saya semakin tumbuh saat saya menjelajahi berbagai situs web di [godly.com](https://godly.website/), banyak di antaranya yang saya anggap sangat keren.

Sebelum terjun ke pengembangan web, saya bekerja sebagai barista di sebuah kafe selama 4 bulan. Ya, jika kita bertemu, saya akan membuatkan secangkir latte untuk Anda. Kembali ke topik sebelumnya, saya belajar pengembangan web secara otodidak melalui YouTube dan Chrome. Pertama kali saya belajar pengembangan web, saya membuat portofolio pribadi tentang diri saya menggunakan HTML, CSS, dan Bootstrap. Pada saat itu, saya belum tahu apa itu React.js, JavaScript, TailwindCSS, dan lain-lain, saya seperti seorang anak yang tidak tahu apa-apa.

Saya belajar React.js melalui platform YouTube, khususnya dari [channel FreeCodeCamp](https://youtube.com/@freecodecamp?si=zq2OWMAlzNhwlfEL), di mana [Bob Ziroll](https://www.linkedin.com/in/bobziroll), kepala pendidikan di [Scrimba](https://v2.scrimba.com/home), menjelaskannya. Saya belajar banyak tentang React.js dan sangat bersyukur kepada guru tersebut. Dia menjelaskan semuanya dengan detail dan jelas, jujur saja.

Jika Anda tertarik dengan React.js, saya merekomendasikan menonton video tersebut atau mencari tutorial di https://www.freecodecamp.org/. Anda bisa menemukan tutorial tentang berbagai topik di channel tersebut, termasuk Python, C++, C, dan banyak lagi.

### Fakta menarik tentang diriku, ayo kita mulai!!

Saya telah membuat beberapa situs web mulai dari portofolio pribadi, situs web untuk sekolah, dan hingga hari ini membangun situs web yang sangat kompleks menggunakan [astro.js](https://astro.build/), Anda dapat melihat beberapa proyek saya di [repository GitHub saya](https://github.com/Raffyshira).

### Fakta unik tentang diriku

Ketika saya membuat sebuah proyek, orang biasanya berpikir saya menggunakan laptop. Tetapi sebenarnya, saya membuat situs web hanya menggunakan android. Ya, jangan terlalu terkejut. Banyak orang masih berpikir bahwa Anda membutuhkan laptop untuk belajar pengembangan web, tetapi saya pikir itu adalah kesalahpahaman. Saya masih bisa membangun situs web yang sangat keren hanya dengan menggunakan android. Ada beberapa alasan mengapa saya tidak menggunakan laptop, salah satunya adalah saya tidak punya cukup uang untuk membelinya. Pada saat itu, saya berpikir bahwa saya membutuhkan laptop untuk belajar pengembangan web, jadi saya mencari cara untuk belajar tanpa itu. Saya ingat sebuah kutipan yang mengatakan:

> Di mana ada kemauan, di situ ada jalan

Hingga saat ini, kemajuan belajar saya telah berkembang dari tidak tahu apa-apa menjadi tahu banyak tentang pengembangan web. Saya telah belajar banyak hal, dan saya berharap perjalanan saya tidak berakhir di sini. Saya akan terus menjelajahi pengembangan web. Hingga saat ini saya benar-benar melihat kemajuan dari awal tidak tahu hingga mengetahui banyak hal.
